# IT Ticket Tracker

## Description
This project is a simple IT Ticket Tracker website built using HTML, CSS, and JavaScript. It allows a user to enter an IT issue and submit it to a list. Each submitted issue is displayed with a **Pending** status badge. The project is designed to be clean, modern, and easy to use.

## Features
- Lets the user type an IT support issue into a text box
- Adds submitted issues to a ticket list
- Displays each ticket with a green **Pending** status badge
- Uses a clean and professional user interface
- Has a responsive layout that works on different screen sizes
- Built in one single file for simplicity

## How It Works

### HTML
The HTML creates the structure of the webpage. It includes:
- A page title
- A text box for entering IT issues
- A submit button
- A list area where tickets are displayed

### CSS
The CSS styles the webpage. It:
- Creates a clean white background
- Styles the button with a blue colour
- Designs the ticket list cards
- Adds spacing, rounded corners, and a modern layout
- Makes the page look professional and easy to read

### JavaScript
The JavaScript adds the functionality. It:
- Reads the text entered by the user
- Checks that the input is not empty
- Creates a new ticket item when the button is clicked
- Adds the issue to the list on the page
- Shows a green **Pending** badge next to each ticket
